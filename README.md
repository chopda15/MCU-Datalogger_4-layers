# MCU Datalogger 4 Layers

KiCad project files for an MCU datalogger board, including the main schematic, PCB layout, project metadata, generated fabrication outputs, PDFs, and local footprint/symbol libraries.

## Contents

- `MCU Datalogger.kicad_pro` - KiCad project file
- `MCU Datalogger.kicad_sch` - main schematic
- `MCU Datalogger.kicad_pcb` - PCB layout
- `footprints and symbols data/` - custom libraries and 3D models
- `pdf/` - exported documentation and drawings
- `MCU Datalogger_4 layer/` and `MCU Datalogger_4 layers/` - output files and repo notes
